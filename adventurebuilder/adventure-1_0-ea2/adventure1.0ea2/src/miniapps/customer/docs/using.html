Customer Mini-Application at a Glance

The Customer mini-application helps you to create a new account in the database or to get the details of an existing account in the database

Using the Mini-Application

This section describes how to use the Customer mini-application

Creating a new account

To create a new account , follow these steps:

  1. On the home page, select Create Account

  2. Enter your account information . All the fields are manadatory

  3. Click Create Account.

    The status will be shown in the next page , along with your account information

Note : This mini-application shares a database table with the Java Adventure Builder Demo. If you have already created a new account with the Adventure Builder Demo , you will not be able to create another account with the same user name. You will get an error message saying that the account already exists and cannot be recreated

Getting the details of an existing account

To get the details of an existing account , follow these steps:

  1. On the home page, select Read Account

  2. Enter your User ID.

  3. Click Read Account.

    The account details will be shown in the next page
