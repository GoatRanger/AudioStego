Customer Mini-Application

This Customer mini-application is a very simple enterprise application that presents an easier learning path to the J2EE platform by focussing on a limited set of applications features. The application helps you to create a new account in the database or to get the details of an existing account in the database .

About the Java BluePrints for the Enterprise program

The Java BluePrints for the Enterprise program defines the application programming model for the J2EE platform. It provides best practice guidelines and architectural recommendations for real-world application scenarios to enable developers to build portable, scalable, and robust applications using J2EE technology.

The Java BluePrints for the Enterprise program guidelines are in the second edition of the book Designing Enterprise Applications with the J2EE Platform. This best-selling book is available at bookstores everywhere, including at the websites DigitalGuru and Amazon.com.

Check out the Java BluePrints Web site at http://java.sun.com/blueprints/ for the latest content.
