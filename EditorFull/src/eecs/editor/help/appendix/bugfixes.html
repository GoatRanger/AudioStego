Changes from version 1.2 to version 1.3

  1. Fixed a bug in the file dialog for Save As... where it wouldn't display the correct default extension
  2. Added a "Preview My Web" that opens the cadet default on Gemini
  3. Added a Deploy option that copies a specified directory to the web directory on Gemini
  4. Fixed the lexical specification for XHTML to correctly identify unary tags and reject attributes that don't have quoted values

Changes from version 1.1 to version 1.2

  1. Fixed Syntax Coloring, automatically restarts if fails during run
  2. Added a "Preview" option for HTML pages to preview the in-memory document without saving

Changes from version 1.01 to version 1.1

  1. Fixed a bug where "Unknown Option" would appear in the status bar when the file was saved.
  2. No longer automatically adds a file extension when saving a file.
  3. Fixed a bug where the compiler wouldn't return errors on a slow computer (thread timing issue)
  4. Fixed a bug that when closing a file that the editor would request a second save of the file (wouldn't remember that the file was just closed)
  5. Fixed the exit editor function so it doesn't call the close file method first (duplicate effort)
  6. Added validation of data from the clipboard to remove illegal characters and reformat "smart quotes" from a word processor into simple quotes (ASCII 34)

Changes from version 0.7 to 1.0

  1. Added a Close function to close the current file without closing the editor. (Renamed the old close to Exit
  2. Many more, don't remember them all!

Changes from version 0.6 to Version 0.7

  1. Fixed the Java code template used when selecting New
  2. Fixed errors in the XHTML code template

Changes from version 0.5 to Version 0.6

  1. Added the ability to enable/disable error logging from the Help Menu
  2. Fixed issues with the text size selection changing alignment between text and line number
  3. Updated syntax coloring to reduce coloring errors caused by invalid lock acquiring sequence

Changes from version 0.4 to Version 0.5

  1. Fixed deadlock issues where editor would freeze due to a race between the Document insertString and the syntax colorer
  2. Added error logging for all major classes
  3. Fixed issues with problem evaluation (Run Tests option) getting incorrect results for tests with multiple outputs
  4. Added a link to the IT105 course handbook on the Help Menu
  5. Added line numbering to the editor pane

Changes from version 0.3 to Version 0.4

Changes from Version 0.22 to Version 0.3

Changes from Version 0.1 to Version 0.22

  2. Fixed Bugs in Gauntlet error checking that caused Compile/Run menu options to be permanently disabled
  3. Added this help functionality
  4. Added the View menu with ability to change text size
